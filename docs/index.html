Script Hub Grandpaniu

给 Shadowrocket 用的 Script Hub 增强工具。可以一键安装增强模块、导入公开模块 URL、分析模块内容，并生成 Shadowrocket 可安装入口。

一键安装到 Shadowrocket 打开小白安装页 查看模块原始文件
主模块Shadowrocket
支持识别QX / Surge / Loon

小白先看这里

第一步先安装增强模块。安装后再用下面的模块库或 URL 检测转换工具。

立即导入 Shadowrocket 备用安装页

如果一键安装没反应，确认手机已安装 Shadowrocket；仍无反应就打开“备用安装页”。

使用顺序

1. 安装增强模块 → 2. 打开 Shadowrocket 并启用模块 → 3. 使用模块库或 URL 检测转换 → 4. 对第三方模块自行判断安全性。
