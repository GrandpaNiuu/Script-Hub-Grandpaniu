安装组件
完整体验需要两个模块：主模块 V2 负责本地分析和转换；Web Addon 负责在指定网页右下角显示模块助手。
1. 主模块 V2
必须安装。安装后可打开 http://grandpaniu.script.hub 进行健康检查。
安装主模块 V2
2. 网页增强附加模块
可选安装。用于在 GitHub、Kelee、ddgksf2013.top 等指定网页里自动出现模块助手。
安装 Web Addon
安装后请在 Shadowrocket 中启用模块，并开启 MITM / HTTPS 解密，证书需要在 iPhone 设置里信任。


  
说明
普通链接怎么用
直接粘贴到“单链接转换”，点“分析/转换”。
网页里自动识别怎么用
安装 Web Addon 后，用 Safari 打开支持域名，如果页面里有模块链接，右下角会出现 Grandpaniu 模块助手。
为什么不做全网改写
全网 MITM 会带来隐私风险、网页变慢、登录异常。这个工具只对指定站点生效。
convertType 怎么选
.sgmodule / .module = surge-module.conf = qx-rewrite.plugin / .lpx = loon-plugin.list = rule-set
