# 路线图

## 近期

- 根据 Script Hub 维护者反馈确认更准确的内置模块接入方式。
- 增加更多真实公开模块样例，覆盖 Loon、Surge、Stash、QX。
- 给增强入口增加更友好的错误提示和复制按钮。

## 后续

- 增加 Web 页面，方便粘贴模块 URL 后直接生成小火箭安装入口。
- 增加规则校验，提示某些模块是否可能无法被 Shadowrocket 直接使用。
- 在发布 release 前自动检查 `docs/authorization.md` 和 `NOTICE.md`。

## 边界

- 不复制 Script Hub 私有代码、私有 API、内置脚本资源或付费内容。
- 未获额外授权前，不声称官方背书或官方内置。
- 授权变化必须记录到 `docs/authorization.md`。
