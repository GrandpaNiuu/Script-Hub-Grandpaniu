本项目是一个独立的兼容性适配器原型，用于识别其他工具的模块、插件或
manifest，并转换为可配置的小火箭安装 payload。

本项目目前不是 Script Hub 官方模块，也不代表 Script Hub 官方立场。

如果未来要将本项目内置到 Script Hub，或以 Script Hub 官方/内置增强模块的
形式发布，维护者应优先取得 Script Hub 相关权利方或维护者的授权。若尚未取
得正式授权，发布页面、README、包元数据和代码仓库中至少应保留明确声明：

- 本项目为独立兼容性工具。
- 本项目不复制、内置或再分发 Script Hub 源码、私有 API、脚本资源或付费内容。
- 第三方模块/插件仅基于用户提供的 manifest 或公开许可信息进行转换。
- 一旦取得授权，应在 `docs/authorization.md` 中记录授权主体、授权范围、日期和
  必要署名。
